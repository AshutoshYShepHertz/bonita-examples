This project illustrates how to use Bonita BPM REST APIs.

Please download a Bonita Tomcat bundle from website http://www.bonitasoft.com/how-we-do-it/downloads
Unzip it on any location on your computer (please avoid spaces or special characters in path)
Start it using <tomcat home>/bin/startup(.sh | .bat)

To start the program execute the following command line:
java -jar rest-api-example-1.0.0-SNAPSHOT.jar
